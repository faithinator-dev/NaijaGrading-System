The **Nigerian Academic Grading System** is a modern, responsive web application designed to help students, educators, and parents calculate and understand academic grades across different educational levels in Nigeria. The system covers three major educational tiers: Secondary School, University, and Polytechnic/Monotechnic institutions.

2. **University (5.0 Scale)**
   - CGPA range: 0.0-5.0

3. **Polytechnic/Monotechnic (5.0 Scale)**
   - CGPA range: 0.0-5.0
   - Classifications: Distinction, Upper Credit, Lower Credit, Pass, Fail
   - Focused on technical and vocational education

## ğŸ“Š Grading Criteria

### Secondary School (0-100)
- **A1**: 75-100 (Excellent)
- **B2**: 70-74 (Very Good)
- **B3**: 65-69 (Good)
- **C4-C6**: 50-64 (Credit)
- **D7-E8**: 40-49 (Pass)
- **F9**: 0-39 (Fail)

### University (5.0 Scale)
- **First Class**: 4.50-5.00
- **Second Class Upper**: 3.50-4.49
- **Second Class Lower**: 2.50-3.49
- **Third Class**: 1.50-2.49
- **Pass**: 1.00-1.49
- **Fail**: Below 1.00

### Polytechnic (5.0 Scale)
- **Distinction**: 4.50-5.00
- **Upper Credit**: 3.50-4.49
- **Lower Credit**: 2.50-3.49
- **Pass**: 1.50-2.49
- **Fail**: Below 1.50
